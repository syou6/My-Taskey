テスラ充電ネットワーク

世界最大の高速充電ネットワークで、どこでも簡単に充電できます。

スーパーチャージャー

長距離旅行に最適な高速充電。わずか15分で最大275kmの航続距離を追加。

自宅での充電

自宅に設置したウォールコネクターで、毎朝フル充電の状態で出発できます。

目的地充電

ホテル、ショッピングセンター、レストランなどで滞在中に充電できる便利な充電スポット。
